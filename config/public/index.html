ReactUnity Previewer

Congratulations on starting your ReactUnity project. You can now test your app inside Unity by setting the DevServer property of ReactRendererUGUI or ReactRendererUIToolkit components to the address of this page.

If your UI is simple, you can also test your UI inside this page with a WebGL template. There are 2 ways you can do that.

Note that as ReactUnity is not optimized for WebGL, you will have a low performance by testing in WebGL.

1 - Install the default previewer

You can install the @reactunity/previewer package for this. 

        yarn add --dev @reactunity/previewer
        # OR
        npm i --D @reactunity/previewer

2 - Use a custom previewer

To do this, you need to build a scene in Unity with ReactRendererUGUI component added. The Type property must be set to Url and Source Path property must be set to /index.js. After building, you must copy the built directory to previewer folder inside your React project, so that there is a previewer/index.html file.
